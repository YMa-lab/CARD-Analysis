Simulation and Analysis

Ying Ma

May 24, 2021

We performed realistic simulations to evaluate the performance of CARD. To do so, we obtained two published datasets: a scRNA-seq data collected on the mouse nervous system from Zeisel et al 2018 and a spatial transcriptomics data collected on the mouse olfactory bulb. We used the scRNA-seq data to construct the expression levels for 260 spatial locations that were measured in the spatial transcriptomics data. The spatial locations were pre-categorized into three main anatomic regions.

Simulate the spatially-resolved transcriptomics count data

rm(list=ls())
library(SingleCellExperiment)
library(pbmcapply)
### load the spatial pattern data, with spatial location included
load("./pattern_gp_label.RData")
### load the single cell RNA-seq data used to generate the spatial count data
load("./sc_eset.forSim.RData")
### set paramaters
irepeat = 1 # repeat ID
imix = 0    # the percentage of noisy locations
n = 30      # number of cells in each spatial location
con1 = c(0.7,0.1,0.1,0.1)  # basaeline cell type proportion in pattern1
con2 = c(0.1,0.7,0.1,0.1)  # basaeline cell type proportion in pattern2
con3 = c(0.1,0.1,0.7,0.1)  # basaeline cell type proportion in pattern3
mix1 = mix2 = mix3 = c(1.0 - imix,imix)  # percentage of non-noiose/noise locations in each pattern
k = 4                      # number of 4 cell types
ct.select = c("Astrocytes","Neurons","Oligos","Vascular") # we selected these fourcell types to be existed in the spatial data
set.seed(irepeat)
### simulate the spatial transcriptomics count data with the funcs.R
source("../funcs/funcs.R")
spatial.pseudo = generateSpatial_norep_fixedProp(
seed = irepeat,
sc_eset = sc_eset, ### the loaded sc_eset
ct.varname = "cellType", #### column name that sepcify the cell type assignment
ct.select = c("Astrocytes","Neurons","Oligos","Vascular"),
sample.withRep = F,
pattern = pattern_gp_label,
ntotal = n,
con1 = con1,
con2 = con2,
con3 = con3,
mix1 = mix1,
mix2 = mix2,
mix3 = mix3,
ncores = 20)
print(rowSums(spatial.pseudo$Sample_random))
### save the simulated data
save(spatial.pseudo,file = paste0("./sim.pseudo.MOB.n",n,".cellType",k,".Mixnoise",imix,".repeat",irepeat,".RData"))
